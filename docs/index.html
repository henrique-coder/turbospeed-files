🚀 TurboSpeed Files

Ultra-fast download links for test files

Quick Download

Use short URLs that work with any download tool:

https://henrique-coder.github.io/turbospeed-files/{SIZE}

Replace {SIZE} with any file size - works with curl, wget, browsers, and all download tools

Example URLs

/100kb → turbospeed-file-100kb.bin
/1mb → turbospeed-file-1mb.bin
/2gb → turbospeed-file-2gb.bin

Perfect For

🌐 CLI Tools

Works perfectly with command line downloaders

curl -O henrique-coder.github.io/turbospeed-files/1mb

📊 Bandwidth Testing

Measure download speeds with known file sizes

wget henrique-coder.github.io/turbospeed-files/100mb

🛠️ API Testing

Test file handling in applications and scripts

turbodl download henrique-coder.github.io/turbospeed-files/10mb
